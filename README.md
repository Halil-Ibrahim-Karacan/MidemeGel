## 🛠️ Teknolojiler

- **Dil:** Kotlin
- **UI:** Jetpack Compose(Material Design 3)
- **Veritabanı:** SQLite
- **Mimari:** MVVM
- **Bağımlılık:** Hilt
- **Yapı:** Clean Architecture, Clean Code
- **Modlar:** Gündüz-Gece Modu
